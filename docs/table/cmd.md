# 运行的一些东西(Win)

`以下操作均为Windows运行`

`Win + R`打开运行窗口，输入以下内容

（在cmd中输入是一样效果）（不区分大小写）

| 命令                   | 打开                           |
| ---------------------- | ------------------------------ |
| Cmd                    |   命令提示符                   |
| Regedit / Regedt32     |   注册表                       |
| Explorer               |   资源管理器                   |
| compmgmt.msc           | 计算机管理（计算机-右键-管理） |
| Calc                   | 计算器                         |
| Taskmgr                | 任务管理器                     |
| services.msc           | 服务                           |
